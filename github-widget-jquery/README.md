# github-widget
用jquery写的github小挂件。这个不能少，毕竟还是很多同学喜欢jQuery的。


# 使用方法
```html
<div class="github-widget" data-username="kyo4311"></div>
<script src="jquery.min.js"></script>
<script src="widget.js"></script>
```

```js
$('.github-widget').githubWidget();
```
