# github-widget
这是一个github小挂件，引入了doT.js模板，还有Q.js可以让两个请求同步。


## 使用方法
```html
<div class="github-widget" data-username="kyo4311"></div>

<div class="github-widget" data-username="olado"></div>

<div class="github-widget" data-username="kriskowal"></div>
```
